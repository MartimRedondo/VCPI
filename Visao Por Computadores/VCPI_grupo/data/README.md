# Datasets

Os datasets devem ser incluídos na pasta `data` do projeto.

# Estrutura de pastas
A estrutura de pastas deve ser a seguinte:

```
data/
├── test_images/
    ├── 00000/
    ├── 00001/
    ├── ...
├── train_images/
    ├── 00000/
    ├── 00001/
    ├── ...
```

Sendo que:
- `test_images/` deve conter as [imagens de teste](https://www.di.uminho.pt/~arf/storage/vcpi/gtsrb/test_images.zip)
- `train_images/` deve conter as [imagens de treino](https://www.di.uminho.pt/~arf/storage/vcpi/gtsrb/train_images.zip).
